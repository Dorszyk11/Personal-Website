# Personal Website - CV

Strona CV stworzona w Next.js z TypeScript i Tailwind CSS.

## Technologie

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm run dev
```

Otwórz [http://localhost:3000](http://localhost:3000) w przeglądarce.

## Build produkcyjny

```bash
npm run build
npm start
```
